#BtMqttSn

[Arduino][Arduino] library for a [MQTT-SN][MQTT-SN] *client* communicating over a [nRF24L01+][nRF24] Transceiver.


### Links
 - [BtMqttSnGateway][BtMqttSnGateway] corresponding [MQTT-SN][MQTT-SN]  *gateway* for [Raspberry Pi][RasPi].
